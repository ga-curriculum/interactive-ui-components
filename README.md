# ![Interactive UI Components](./assets/interactive-ui-components-header.png)

## About

In this module, students will explore the power of CSS to create interactive and dynamic user interfaces, mastering techniques from basic transformations to complex animations. Through practical exercises, they will learn to manipulate HTML elements using CSS transforms, transitions, and animations, ensuring smooth and visually engaging web experiences.

## Prerequisites

- Intro to HTML
- Intro to CSS
- Intermediate CSS
- Intro to JavaScript
