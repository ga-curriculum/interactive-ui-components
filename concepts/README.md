## Why Use Transitions and Animations?
CSS transitions and animations are a way to smoothly change the style of an element over a specified duration. They enhance the user experience by adding subtle and visually pleasing effects to web pages. 

The possibilities are limitless with transitions and animations. You can make a button change color gradually when you move your mouse cursor over it, or you can create loading animations to keep the user engaged while your site is loading.

Such design elements enhance the user experience by making websites feel more interactive and engaging, and can do so without overloading your site with tons of node packages and complex JavaScript.

## Transform, Transition, and Animation CSS Properties

There are three ways to use CSS to animate an element:
1. **`Transform`** - Is a *static* and *immediate* change where your element is warped in some way from its normal state to an altered state. It will then stay in the altered state forever.
1. **`Transition`** - You determine the start and end state, the computer handles the tweening (the states in between the beginning and the end).
1. **`Animations`** - You determine as many states as you want (beginning, end and in-between) - the computer still tweens between those steps. 

Each of these properties enables you to create interactive UI components and should be used strategically to create an improved user experience. Let's explore each one.
